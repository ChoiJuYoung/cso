ⓐ 센스 

  복잡한 맵일수록 증가합니다. 동족전, 장기전에서 유리한 전투를 하도록 도와줍니다.

( ex. 신동원은 센스가 높으니 동족전에서 유리하겠군! )

 

ⓑ 전략

  복잡한 맵일수록 증가합니다. 전략적인 플레이의 성공률이 증가하며, 맵의 상성을 극복하거나 이용할 수 있게됩니다.

( ex. 신동원은 전략이 다른 능력치에 비해 낮으니 상성인 테란에 조금 약하겠군. )

 

ⓒ 컨트롤

  교전마다 소량의 병력손실을 줄여줄 뿐 아니라, 일부 유닛간의 교전에서는 승패에 큰 영향을 미칩니다.

( ex. 신동원은 컨트롤이 높으니 일부 유닛간의 교전이라면 저저전에서의 뮤탈 싸움도 포함되지 않을까?

그렇다면 동족전에서 많이 유리하겠군. )

 

ⓓ 물량

  병력생산, 확장력, 자원관리 등을 모두 포함합니다.

( ex. 신동원은 물량이 높은 편에 속하는 것이겠지? 그렇다면 자원이 많은 맵에서 조금 유리하겠군. )

 

ⓔ 공격력

  러시거리가 가까운 맵에서 증가합니다. 공격할 때 상대에게 더 큰 데미지를 입힐 수 있습니다.

( ex. 신동원은 공격력도 높은 편이야. 러시거리가 가까운 맵에서는 확실히 이길 확률이 높겠는걸? )

 

ⓕ 수비력

  러시거리가 먼 맵에서 증가합니다. 수비할 때 입는 피해를 줄여주며, 적에게 더 큰 데미지를 가합니다.

( ex. 신동원은 수비력도 높은 편이야. 그 말은 러시거리가 먼 맵에서도 이길 확률이 적지는 않다는 거겠지. )

 

ⓖ 견제

  게릴라 플레이의 위력이 증가하며, 중후반 이후 빠른 기동성으로 적의 멀티를 공략할 수 있게 됩니다.

( ex. 신동원은 견제가 높은걸? 장기전으로 갔을 때 좋은 점이 많을거야. )

 

ⓗ 정찰

  게릴라 플레이를 조기에 차단하는 확률이 증가하며, 중후반 이후 적 병력의 움직임을 잘 파악할 수 있게 됩니다.

( ex. 신동원은 정찰이 그렇게 많이 높지는 않아. 전략에 당하거나 장기전으로 갔을 때 병력의 움직임을 파악 못 해 아쉬운 점이 있을 것 같아. )
